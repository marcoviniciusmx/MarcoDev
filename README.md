## 🏠 About the project

Este projeto foi desenvolvido como uma landing page profissional para divulgar os serviços do **MarcoDev**, com foco em apresentar soluções digitais modernas, transmitir autoridade e converter visitantes em oportunidades de negócio.

A proposta do site é destacar serviços como **criação de landing pages**, **automação com inteligência artificial**, exibição de **projetos já desenvolvidos**, apresentação do **processo de trabalho** e oferta de um canal direto para orçamento e contato.

O layout foi pensado para unir **impacto visual**, **navegação fluida**, **responsividade** e uma comunicação estratégica voltada para negócios que desejam fortalecer sua presença digital. 

---

## 🧰 Features

- Header fixo com navegação por âncoras
- Hero section com proposta de valor e chamadas para ação
- Seção “Sobre Mim” com apresentação profissional
- Cards com diferenciais e benefícios do serviço
- Planos de landing pages com precificação
- Seção dedicada a serviços de IA
- Área de resultados com números animados
- Portfólio com projetos desenvolvidos
- Seção de processo de trabalho em formato de timeline
- Área de contato com informações profissionais
- Formulário funcional com integração externa
- Botões estratégicos para orçamento e navegação
- Menu mobile responsivo
- Scroll suave entre seções
- Animações de entrada ao rolar a página
- Partículas animadas na hero section
- Pequenos efeitos de parallax e interações visuais
- Melhorias básicas de acessibilidade com suporte a teclado

## 👷 Project structure

A estrutura principal da landing page está organizada em seções como:

- Início
- Sobre
- Serviços
- Serviços de IA
- Resultados
- Portfólio
- Processo de trabalho
- Contato

Além disso, o projeto conta com separação simples entre os arquivos principais:

- `index.html`
- `style.css`
- `script.js`
- `assets/` para imagens e elementos visuais 

---

## 🎨 Layout and design decisions

O projeto utiliza uma identidade visual com estética tecnológica e futurista, baseada em tons de azul escuro, azul neon e fundos com gradientes, buscando transmitir inovação, performance e modernidade.

Alguns pontos de destaque no layout:

- paleta visual inspirada em tecnologia e alta performance
- tipografia com aparência moderna e forte presença visual
- uso de efeitos de brilho, gradientes e transparências
- cards com bordas suaves e visual interativo
- hierarquia visual bem definida nas seções
- foco em legibilidade, contraste e experiência do usuário
- adaptação para diferentes tamanhos de tela
- elementos animados para enriquecer a percepção de valor do projeto
